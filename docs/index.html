QuotationV3 🚀

Quotation Management System Documentation

Overview

QuotationV3 is a complete CRUD-based quotation management application. It allows users to

  • Create Quotations
  • Add Multiple Items
  • Edit Header Details
  • Edit/Delete Items
  • View All Quotations
  • Sort Quotations by Latest Activity
  • Print Quotations

Edit & Delete Items

Edit Use the Edit button to modify any incorrect item.
Delete Remove unwanted items using the Delete button.
Edit Header Modify Ledger Name, GSTIN or Invoice Date using the Edit button.

Header Menu

🆕 Create

Opens the quotation creation page for creating a new quotation.

📋 ShowAll

Displays all quotations with Ledger Name, Invoice Date and GSTIN.

⬇ Descend

Shows quotations sorted by latest activity. TimeSpan displays

  • 2 min ago
  • 10 min ago
  • 1 hour ago
  • Yesterday

🖨 Print

Opens browser print preview. Users can print or save as PDF.

Workflow

Create Quotation ↓ Fill Ledger Details ↓ Save ↓ Item Entry ↓ Add Item ↓ Save ↓ Add More Items ↓ Edit/Delete Items ↓ Edit Header ↓ ShowAll ↓ Descend ↓ Print
